# NodeJS Hello World
NodeJS Hello World tutorial

## Install
- Download & install https://nodejs.org/en/download
- Checking verions
    ```
    node -v
    npm -v
    ```

## Getting Started
1. create folder `hello-world`
2. init app & input app info
    ```
    npm init
    ```
3. create `index.js` and code a line
    ```
    var msg = 'Hello World';
    console.log(msg);
    ```
4. run app
    ```
    node index.js
    ```
